This folder contains all the necesaary data for this paper. 

YelpZip and YelpNYC folders contain the pre-processed review content, metadata and lables. The sub-folder in each i.e. "Resulting Features" contains features obtained after feature extrction phase. (NOTE: Original YelpZip and YelpNYC datasets are available at http://odds.cs.stonybrook.edu/yelpzip-dataset/ and http://odds.cs.stonybrook.edu/yelpnyc-dataset/ , respectively.)

Opinion-lexicon-English folder contains the files used in feature extraction to find out negative and positive sentiments. (NOTE: This opinion lexicon is available at https://www.cs.uic.edu/~liub/FBS/sentiment-analysis.html)

Results folder contains all the results obtained after model training phase for all three types of features i.e. Behavioral, Textual and Hybrid under all three settings i.e. reviewer-centric, review-centric and product-centric on both the datasets YelpZip and YelpNYC.
